# 🤖 ML Churn Prediction Bot

Бот-интерфейс для модели машинного обучения, которая предсказывает отток клиентов на основе логистической регрессии.

## 🌟 Основные функции
* **Предсказание в реальном времени:** Введите данные клиента, и бот выдаст вероятность оттока.
* **ML Backend:** Используется модель логистической регрессии, обученная на [указать датасет].
* **Удобство:** Вместо запуска кода в Jupyter Notebook, пользователь получает доступ к аналитике через мессенджер.

## 🛠 Технологический стек
* **Язык:** Python 3.10
* **Библиотеки бота:** `aiogram` (или `telebot`)
* **ML-библиотеки:** `scikit-learn`, `pandas`, `numpy`
* **Окружение:** `python-dotenv` (для защиты токенов)
